# 🔗 P2P传输 - 跨网络文件与文本传输工具

基于 WebRTC 的浏览器端点对点传输工具，支持在不同网络环境下的设备之间直接传输文件和文本。

## 🌐 在线使用

**https://hahq.github.io/p2p-transfer/**

手机和电脑都可以直接访问这个地址使用。

## ✨ 功能特点

- 📁 **文件传输** - 支持任意类型、任意大小的文件传输
- 📝 **文本传输** - 快速发送文本内容，支持一键复制
- 🌍 **跨网络** - 手机用流量、电脑用WiFi，不在同一局域网也能传输
- 🔒 **点对点** - 数据直接在设备间传输，不经过服务器
- 📊 **进度显示** - 实时显示文件传输进度
- 🌙 **深色模式** - 支持浅色/深色主题切换
- 📱 **响应式** - 适配手机和电脑屏幕

## 📖 使用教程

### 第一步：打开网页

在两台设备（如手机和电脑）上都打开 https://hahq.github.io/p2p-transfer/

### 第二步：创建房间

1. 在其中一台设备上，点击 **"创建房间"** 按钮
2. 系统会生成一个 **6位房间码**（如：ABC123）
3. 点击 **"复制房间码"** 将房间码分享给对方

### 第三步：加入房间

1. 在另一台设备上，切换到 **"加入房间"** 选项卡
2. 输入收到的 **6位房间码**
3. 点击 **"加入"** 按钮

### 第四步：开始传输

连接成功后，界面会切换到传输界面：

#### 传输文件
- **方式一**：直接将文件拖放到虚线框内
- **方式二**：点击虚线框选择文件
- 支持同时选择多个文件
- 可以查看传输进度
- 接收方点击 **"下载"** 保存文件

#### 传输文本
- 在文本框中输入内容
- 点击 **"发送文本"** 按钮
- 接收方可以点击 **"复制"** 复制文本

### 第五步：断开连接

传输完成后，点击右上角的 **"断开连接"** 按钮即可结束会话。

## 🎨 主题切换

点击页面右上角的 🌙（月亮）图标可以切换深色模式，再次点击 ☀️（太阳）图标切换回浅色模式。

系统会记住您的主题偏好。

## 🔧 技术架构

- **前端**：纯 HTML + CSS + JavaScript，无需构建
- **P2P连接**：WebRTC + PeerJS
- **信令服务**：PeerJS 公共云服务器
- **部署**：GitHub Pages

## ⚠️ 注意事项

1. **浏览器兼容**：推荐使用 Chrome、Edge、Firefox 等现代浏览器
2. **网络环境**：某些严格的防火墙可能会阻止 P2P 连接
3. **连接稳定性**：PeerJS 公共服务器可能偶尔不稳定，如遇连接问题请刷新重试
4. **大文件传输**：传输大文件时请保持页面打开，不要切换到其他应用

## 🚀 本地运行

如果想在本地测试：

```bash
# 进入项目目录
cd p2p传输

# 启动本地服务器（需要Python）
python -m http.server 8080

# 在浏览器打开
# http://localhost:8080
```
